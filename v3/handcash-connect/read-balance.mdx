---
title: Read Balance
description: 'Retrieve balances and spendable limits in v3'
---

## Balances

```typescript
import { getInstance, Connect } from '@handcash/sdk';

const sdk = getInstance({ appId: 'YOUR_APP_ID', appSecret: 'YOUR_APP_SECRET' });
const client = sdk.getAccountClient('<AUTH_TOKEN>');

const { data } = await Connect.getBalances({ client });
for (const b of data?.items ?? []) {
  console.log(b.currency?.code, b.units, b.fiatEquivalent?.units);
}
```

## Spendable balances (app limits)

```typescript
const { data } = await Connect.getSpendableBalances({ client });
for (const s of data?.items ?? []) {
  console.log(s.currencyCode, s.spendableBalance);
}
```
